# FYP Navigation Benchmark Project

This repository is a curated upload bundle for the mobile robot navigation FYP project.

Included components:
- `ws_livox/src/LIO-SAM-MID360`
- `ws_livox/src/fyp_utils`
- `ws_livox/src/mid360_navigation`
- `fastlio2_ws/src/FAST_LIO`
- `fastlio2_ws/src/Point-LIO`
- `fasterlio_ws/src/faster-lio`

Large datasets and generated assets such as rosbags, videos, and PCD maps are intentionally excluded.
