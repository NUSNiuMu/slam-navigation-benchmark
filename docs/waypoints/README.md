# Waypoint Examples

This directory stores example waypoint YAML files that are useful for documentation and evaluation demonstrations.

## Files

- `nav_waypoints_example.yaml`
  - minimal example showing the waypoint file structure
- `nav_waypoints_0324.yaml`
  - a recorded waypoint set used in project experiments

These files are kept at the top-level documentation layer so they remain easy to discover even though the internal `fyp_utils/docs` working directory is excluded from the GitHub repository.
